# DQX Auto Login Launcher

顔タイプ1ピンクほっぺお手をどうぞ中身不一致エル子たちが活躍するゲームの自動ログインを行う非公式ランチャーです。

## 動作環境
* Windows 10 / 11（64bit）
* Ver 1.0〜7.0 がインストール済みであること
* 公式ランチャーで一度以上ログイン済みであること
  ※プレイヤー登録していない場合はゲストモードで起動します

## 注意事項
* 非公式ツールのため、利用によるアカウントへの影響（BANリスク等）は自己責任で使用してください
* `config.json` にはログイン情報が保存されるため他の人に渡さないでください

## 使用方法

1. `launcher.zip` をダウンロードして解凍してください。
2. 解凍したフォルダ内の `launcher.exe` を実行してください。
3. **初回起動時**は画面の指示に従いSQEX ID・パスワード等を入力してください。設定は `config.json` に保存され、次回以降は自動でログイン・起動します。

> Windows Defenderにウイルスとして誤検知される場合があります。不安な場合はソースコードを確認のうえ `launcher.py` を直接実行するか、自身でexe化してください。

## OTPシークレットキーの取得について
* Proton AuthenticatorやAegisの場合、登録しているSQEX用の認証設定から「秘密鍵」や「Secret」に記載されている32文字のキーを確認してください。
* Google Authenticatorはシークレットキーをアプリ外に出す手段が原則ないので、手動入力（毎回6桁を直接入力）を選択するか、前述のアプリに移行してください。

## 設定項目 (config.json)
| キー | 内容 |
| :--- | :--- |
| `user_id` | SQEX ID |
| `password` | パスワード |
| `otp_auto` | OTP自動入力の有効/無効 |
| `otp_secret` | OTPシークレットキー（自動入力が有効な場合のみ保存） |
| `account_token` | SSOトークン（自動保存） |
| `player_number` | 起動するプレイヤー番号（自動取得・保存） |
| `window_w / h` | ゲスト起動用ウィンドウサイズ（自動保存） |
| `window_x / y` | ゲスト起動用表示座標（自動保存） |
| `check_update` | 起動時のアップデート確認（true/false） |

## 機能概要
ログインサーバーに対してID・パスワード・OTPを送信し、取得したSessionIDを使ってゲームを直接起動します。初回ログイン後はSSOトークンを保存して次回以降の認証を高速化します。OTPはTOTP（RFC 6238）準拠で、シークレットキーはローカルにのみ保存され通信には使用されません。

ゲームのインストールパスはレジストリから自動検出します。また、ゲーム本体が保持する難読化済みのプレイヤーリストファイルを復号して読み取り、ゲーム起動時に適切なプレイヤー番号を引数として渡します。

### ウィンドウ制御（ゲストモード時のみ）
プレイヤー番号が特定できない場合（ゲストモード起動時）に限り、ウィンドウサイズと位置を管理します。ゲストモードでは公式ランチャーによるウィンドウ設定が引き継がれないため、前回終了時のサイズ・座標を記録して次回起動時に復元します。

## トラブルシューティング
* 起動後にログインエラーが表示される場合、公式ランチャーを起動してゲームのアップデートがないか確認してください。
* その他問題が発生した場合は、ツールと同じフォルダにある `.log` ファイルを確認してください。エラーの詳細が記録されています。<br>※まずはconfig.jsonの削除、公式ランチャーでの通常起動を行った後、初回起動からやり直してください。
